    Roadmap - Syn.Bot

    The following features maybe considered for a future version of Syn.Bot Framework.

    OSCOVA

    Feature Description
    Expansion Library Expansion pack for large and complex entity recognizers.
    Test Attribute InBuilt unit testing for expressions invoking right intents.
    Linked Intent Support Execute multiple intents if their interpretation don't overlap.
    Events or Activity Add support for Event triggers on Request
    Import & Export Enable importing and exporting of components (Intents, Entities)
    Previous intent Use previously executed intent's entities for contextual intents
    EvaluateJson Evaluate a serialzed json request. Requires Serialize and Deserialize methods in Request class

    Rejected Features

    • Unique annotation of example entities only. - Leads to conflicts in Expression declaration.

    SIML

    Feature Description
    Cached suggestion Cache previously generated Suggestion result based in Search Query.
    Extend Load(XElement) in Settings Support <Siml> as root Element in Load() methods.
    SimlOptimizer class Helps in optimization of SIML documents.
    Status or Trace Detailed generation of logs for internal working of the Bot and its adapters.

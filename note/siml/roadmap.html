Roadmap - SIML

The following features maybe considered for a future version of SIML specification.

Feature Description
Sense or Search Adapter Replacable In-Built input matching system
Goto Timing Allows reiteration of input commands at a given interval.
Late Adapter Generate response after the specified delay interval.
Event Adapter A Clone of the Model type however doesn’t yield a response immediately, instead triggers an event that has to be handled by an event handler. Uses the adapter to explicitly raise an event via an SIML Model.

Features in Detail

Sense/Search

A new tag called Sense which can be used of the Pattern tag. The point behind using Sense is to evaluate user input based on accuracy.

  • Loose pattern matching
  • The functionality of the Sense parser can be modified by developers.
  • Will support Sets and Regex
  • Important words can be marked using the tilde sign

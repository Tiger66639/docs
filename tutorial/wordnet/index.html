    WordNet

    WordNet is a lexical database developed by Princeton University where nouns, verbs, adjectives and adverbs are organized into synonym sets, each representing one underlying lexical concept.

    WordNet® is a registered trademark of Princeton University.

    Syn WordNet

    Syn.WordNet is a portable class library designed to read WordNet lexical database files via StreamReader. Besides, a WordNetEngine the library also provides SemanticSimilarity class that assists in finding semantic similarities between words and sentences.

    NuGet

    You can import Syn.WordNet in your .NET/Xamarin or Mono projects via NuGet Package Manager. Just type the following command to reference Syn.WordNet in your project.

    PM> Install-Package Syn.WordNet

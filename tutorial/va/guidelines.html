    Guidelines

    Plugin Names

    The name of all plugins should start with Syn.VA.Plugins. The General plugin loads all dynamic link libraries in the Plugins folder that start with the name Syn.VA.Plugins. This is to prevent loading of dependency libraries.

    So if you have a Plugin is about a Robot then the proper plugin namespace and assembly name would be Syn.VA.Plugins.Robot....

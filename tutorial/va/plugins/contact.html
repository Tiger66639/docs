    Contact Plugin

    Enables fetching of Windows Contact details like phone numbers and email addresses using contact names.

    SIML Adapters

    Contact

    Annotation <o:Contact>
    Definition Gets the contact number or the email address of the specified contact.
    Children MUST contain an arbitrary text specifying the contact name and MUST contain a Get attribute with value Number or Email

    Usage

    <o:Contact Get="Number">Peter</o:Contact>

    Returns the phone number of Peter

    <o:Contact Get="Email">Peter</o:Contact>

    Returns the address of Peter

    Sets

    Name Description
    Contact Names of all the Windows Contacts stored.

    Requirements

    • Contact list in Windows Contact

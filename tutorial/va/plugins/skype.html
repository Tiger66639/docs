    Skype Plugin

    This plugin lets users make a skype call using Virtual Assistant commands.

    If Skype is not initially launched by the Virtual Assistant then users must again login after using the command is activated.

    SIML Adapters

    Skype

    Annotation <o:Skype>
    Definition Makes a Skype call to the specified User
    Children MUST contain an arbitrary text specifying Skype User's ID

    Usage

    <o:Skype>echo123</o:Skype>

    Calls echo123 in Skype.

    Skype must be installed in user computer for this adapter to work.

    Settings

    Name Description
    Skype-Users List of all known Skype contact names.

    MovieInfo Plugin

    This plugin allows the user to fetch details about a movie using Omdb API interface.

    SIML Adapters

    MovieInfo

    Annotation <o:MovieInfo>
    Definition Gets information regarding the movie name specified.
    Children Must contain an arbitrary text specifying the name of the movie and a Get attribute that specifies the type of information to fetched.

    Attribute

    Name Description
    Get The type of information to retrieve. Values can be Plot, Director, Rating, Language, Genre, Votes, Writer, Country, Year, Awards and Actors

    Usage

    <o:MovieInfo Get="Director">Life is beautiful</o:MovieInfo>

    Returns the name of the director of the movie "Life is beautiful".

    <o:MovieInfo Get="Plot">Contact</o:MovieInfo>

    Gets plot of the movie "Contact".

    This adapter returns a response in Text format.

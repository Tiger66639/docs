    Wolfram Plugin

    Provides an adapter to use the Wolfram Alpha API feature.

    SIML Adapters

    Alpha

    Annotation <o:Alpha>
    Definition Gets Html content returned from Wolfram Alpha API call for the specified query.
    Children Must contain an arbitrary text specifying the search query.

    Usage

    <o:Alpha>Jupiter</o:Alpha>

    Searches Wolfram Alpha for Jupiter

    Events

    Name Description
    Alpha-No-Key Raised when <o:Alpha> is evaluated with no API key.
    Alpha-Result Raised when <o:Alpha> is evaluated and a result is generated. The variable Alpha-Content will contain the returned HTML content.
    Alpha-No-Result Raised after <o:Alpha> is evaluated but no result is generated. The variable Alpha-Query will contain the search query.

    Settings

    Name Description
    Api-Key Wolfram Alpha Developer API Key.

    Requirements

    • An active Internet connection.
    • Wolfram Alpha Developer API Key.

    Launcher Plugin

    Command Format:

    • Open [Application] - Launches the specified application.
    • Open [Folder] Folder - Opens the specified special folder.
    • Open [Website] - Opens the specified website.
    • Search [Search-Engine] for [Query] - Performas a query search on specified search engine.

    Options

    • [Application]: Application name.
    • [Folder]: Special folder name.
    • [Website]: Website name.
    • [Search-Engine]: Search engine name.
    • [Query]: Search query

    Example Command:

    • Open **Notepad**
    • Open Music folder
    • Open Google
    • Search Google for Syn Virtual Assistant

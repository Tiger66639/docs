    Syn Bot Assistant

    The Syn.Bot.Assistant library has been designed for rapid bot deployment on ASP.NET websites and Telegram. The library provides tailored assistant classes that can be used deploy bot in minutes.

    At the moment the library relies on .NET Framework version 4.5 and supports Windows and Linux environments.

    NuGet Package

    PM>Install-Package Syn.Bot.Assistant

    Platforms Supported

    Requirements

    • .NET Framework on Windows environment.
    • Mono on Mac OS or Linux environment.
